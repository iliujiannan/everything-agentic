# 技术栈

## 后端

- 语言 / 框架：`${BACKEND_LANG}` + `${BACKEND_FRAMEWORK}`
- 构建工具：`${BUILD_TOOL}`
- 依赖管理：`${DEPENDENCY_MANAGER}`

## 数据库

- 类型 / 版本：`${DB_TYPE}` `${DB_VERSION}`
- 驱动：`${DB_DRIVER}`
- 连接配置：`${DB_TIMEZONE}`
- DDL 迁移工具：`${MIGRATION_TOOL}`，命名规则：`${MIGRATION_NAMING}`
- 注意事项：`${DB_NOTES}`

## 代码质量

- 扫描工具：`${SONAR_TOOL}`（如 SonarQube Cloud / SonarQube Server）
- 项目 Key：`${SONAR_PROJECT_KEY}`
- Quality Gate：`${SONAR_QUALITY_GATE}`

## 测试

- 单元测试框架：`${TEST_FRAMEWORK}`
- Mock 库：`${MOCK_LIB}`
- 集成测试：`${INTEGRATION_TEST}`（如 Testcontainers + 对应镜像）

## 容器化

- 镜像构建：Docker
- 编排平台：Kubernetes
- Registry：`${CONTAINER_REGISTRY}`
- 命名空间：`${K8S_NAMESPACE}`

## CI/CD

- 工具：`${CICD_TOOL}`
- 触发策略：`${CICD_TRIGGER}`

## 前端（如有）

- 框架：`${FRONTEND_FRAMEWORK}`
- 构建工具：`${FRONTEND_BUILD}`
- 包管理：`${PKG_MANAGER}`
