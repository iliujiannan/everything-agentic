## 5. 分页查询机制

| 请求参数 | 类型 | 说明 |
|---------|------|------|
| pageNumber | String | 页码，从 1 开始 |
| pageSize | String | 每页数量，建议 50 |

**自动翻页终止条件（`requestAllPage()`）：**

1. 接口返回 `code != 200` → 抛出异常
2. 本页返回数量 `< pageSize` → 已到最后一页
3. 本页返回为空
